# Med-Synthetix

Med-Synthetix is an autonomous, agentic AI research assistant designed to bridge the gap between patient diagnostics and experimental clinical trials. 

## 🚀 The Problem
Medical researchers and doctors spend hours manually cross-referencing complex patient medical histories against hundreds of clinical trial inclusion/exclusion criteria. This manual process is time-intensive and prone to error.

## 💡 The Solution
Med-Synthetix acts as an **Agentic Medical Auditor**. It autonomously:
* Ingests anonymized patient records and clinical trial protocols.
* Uses a "Critic" agentic workflow to validate matches against specific criteria.
* Provides verifiable source citations for every recommendation, ensuring transparency and trust.

## 🛠️ Tech Stack
* **AI Engine:** Google Gemini 1.5 Flash
* **Frontend:** HTML5, Tailwind CSS
* **Orchestration:** Agentic Logic (Gemini Canvas)
* **Deployment:** Vercel

## 📋 How to use
1. Enter your Gemini API Key in the top-right field.
2. Upload a Patient Summary PDF and a Clinical Trial Protocol PDF.
3. Click **"Analyze Patient-Trial Match"** to generate the compatibility scorecard.
